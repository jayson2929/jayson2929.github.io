  2. Command Line Cheat Sheet

    • pwd print wokring directory: outputs the name of the current working directory.
    • ls list files: lists all files and directories in the working directory.
    • cd change directory: switches you into the directory you specify.

      • +) cd ..

      +) cd ../..

    • mkdir make directory: creates a new directory in the working directory.
    • touch (verb): create, change and modify timestamps of a file
    • clear (verb): clears the terminal
